# 📊 Data Analytics Project – Customer Shopping Behavior Analysis

## 📌 Overview

This project demonstrates a complete end-to-end Data Analytics workflow using Python, SQL, MySQL, and Power BI. The objective is to analyze retail customer behavior, identify shopping trends, and present business insights through an interactive dashboard and report.

The project follows a real-world analytics process from data loading and cleaning to visualization and reporting.

---

## 📂 Dataset

**Dataset:** Retail Customer Behavior & Shopping Trends

The dataset contains customer information such as:

- Customer Details
- Age & Gender
- Product Category
- Purchase Amount
- Payment Method
- Location
- Frequency of Purchase
- Customer Review
- Subscription Status

## 📈 Project Workflow

### 1. Load Dataset
- Imported the dataset into Python.
- Explored rows, columns, and data types.

### 2. Exploratory Data Analysis (EDA)
- Checked missing values.
- Generated summary statistics.
- Analyzed customer behavior.
- Visualized important trends.

### 3. Data Cleaning
- Removed duplicate records.
- Handled missing values.
- Corrected data types.
- Prepared clean data for analysis.

### 4. SQL Analysis (MySQL)
- Imported cleaned data into MySQL.
- Wrote SQL queries to analyze:
  - Total Sales
  - Customer Distribution
  - Product Performance
  - Purchase Trends
  - Payment Methods
  - Category-wise Sales

### 5. Power BI Dashboard
Created an interactive dashboard including:
- Total Sales
- Total Customers
- Product Category Analysis
- Gender Distribution
- Purchase Trends
- Location Analysis
- Payment Method Analysis
- Customer Subscription Status

### 6. Report & Presentation
- Prepared a business report summarizing key insights.
- Created a PowerPoint presentation to present findings.

## ▶️ How to Run

1. Clone this repository.
2. Install the required Python libraries.
3. Run the Python notebooks for data loading, cleaning, and EDA.
4. Import the cleaned dataset into MySQL.
5. Execute the SQL queries.
6. Open the Power BI `.pbix` file to explore the dashboard.
7. Review the project report and presentation.
